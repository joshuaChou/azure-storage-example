# Introduction
 - platform:*node version 8.x*

# Getting Started

> This example is  test azure storage for emoana

`npm install` 

### Init config 

`npm run init`

> set your connection string in config.ts 

### show help command

`npm run help` 

### start example

`npm start`

# important 

   > azure storage key is only from test , if use in production must renew and set privatey rule.
